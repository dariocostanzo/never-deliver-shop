# Media Assets

Drop your project visuals in this folder using these filenames so `README.md` renders them automatically:

- `home.png`
- `product-detail.png`
- `checkout.png`
- `search-suggestions.gif`
- `focus-states.gif`

Recommended:

- PNG screenshots: around 1400px wide
- GIF demos: keep under 8 MB when possible
- Use compressed assets for faster GitHub page load
